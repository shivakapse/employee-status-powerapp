# 📋 Employee Weekly Status Power App

A Canvas Power App built for weekly employee status tracking with secure OTP login,
structured reporting, and real-time Power BI dashboard visualization.

---

## 🎯 Business Problem

Teams had no structured way to report weekly progress. Managers lacked
visibility into blockers and goals — leading to delayed decisions and
missed escalations.

---

## ✅ Solution Built

A full end-to-end Canvas Power App that allows employees to:
- Log in securely via OTP authentication
- Submit weekly status updates in a structured form
- View team progress via an embedded Power BI dashboard

---

## 🔧 Features

| Feature | Details |
|---|---|
| 🔐 Login Screen | Secure entry point with OTP authentication |
| 📝 Weekly Status Form | What was achieved, next week goals, blockers |
| 🚨 Blocker Tracking | Flags support needed for manager visibility |
| 📊 Power BI Dashboard | Visual summary of team weekly status |
| ⚡ Power Automate | Automated notifications and data flow |

## 📐 App Architecture
```
Login Screen (OTP Auth)
        ↓
Weekly Status Form
  ├── What did you achieve this week?
  ├── What are your goals for next week?
  └── Any blockers or support needed?
        ↓
SharePoint / Dataverse (Data Storage)
        ↓
Power Automate (Notifications to Manager)
        ↓
Power BI Dashboard (Team Status Visualization)
```
